# SPA Task application (Django, React)


### This application create for SpurIT
> ### Technologies used:
>
> - Docker & Docker-Compose
> - Django
> - Django Rest Framework
> - React
> - MUI
> - JWT

## Instalation

### 1. Start application
```bash
make fup
```

### 2. Make database migrations
```bash
make migrate
```

## 3. Run UI application
```bash
cd frontend/ && npm install && npm run serve
```
